Career Path Skill Recommendation helps users find essential skills for different career paths.

== Description ==
This plugin allows users to select a career path and receive relevant skill recommendations.  
Features:
- Career path selection  
- Dynamic skill recommendations  
- Admin panel to manage career paths  
- Shortcode `[career_skills_display]`  

== Installation ==
1. Upload the plugin to `/wp-content/plugins/`
2. Activate via WordPress Dashboard
3. Add `[career_skills_display]` to a page  

== Frequently Asked Questions ==
**Can I add more career paths?**  
Yes, use the admin settings page.  

**Does it support translations?**  
Yes, `.pot` file included.
